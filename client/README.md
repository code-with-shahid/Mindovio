# Mindovio Client

Frontend for **Mindovio** — AI Powered Smart Study Assistant.

## Scripts

- `npm run dev` — start Vite dev server
- `npm run build` — production build
- `npm run preview` — preview production build
- `npm run lint` — ESLint

See the root [README](../README.md) for `VITE_API_URL` and Firebase configuration.
